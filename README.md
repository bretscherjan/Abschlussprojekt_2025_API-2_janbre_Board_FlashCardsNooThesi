# FlashCards NooThesi

Welcome to the FlashCardsNooThesi project! This is a desktop application for creating and using digital flashcards to help you study and learn.

## What it does
- Create your own flashcards with questions and answers.
- Study your flashcards in an easy-to-use interface.
- Import and Export your Flashcards.
- Work together on the same Deck.

## Technologies used
- C#, WPF, .Net in the Frontend
- PHP, MySQL in the Backend
